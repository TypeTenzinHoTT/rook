## Install

<Steps>
<Step title="Install the CLI">

```bash
npm install -g rook-rpg
```

</Step>

<Step title="Initialize Rook">

Run the setup wizard to authenticate with GitHub and connect your repos:

```bash
rook init
```

This will:
1. Prompt for your GitHub Personal Access Token
2. Register you with the Rook backend
3. Let you select repos to connect via webhooks

<Tip>Generate a PAT at [GitHub Settings > Developer settings > Personal access tokens](https://github.com/settings/tokens) with `repo` and `admin:repo_hook` scopes.</Tip>

</Step>

<Step title="Start coding">

That's it. Push commits, open PRs, review code, and close issues. Rook tracks everything automatically.

Check your progress:

```bash
rook stats        # Level, XP, streak, achievements
rook dungeon      # Daily and weekly quests
rook leaderboard  # See where you rank
```

</Step>
</Steps>

## What earns XP?

| Activity | XP |
|----------|-----|
| Commit | 10 |
| Large commit (100+ lines) | 25 |
| PR created | 20 |
| PR merged | 50 |
| PR reviewed | 15 |
| Issue closed | 15 |

Bonuses stack from quest streaks, guild multipliers, and prestige perks.
