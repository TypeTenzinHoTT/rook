## What is Rook?

Rook is a CLI tool that gamifies your GitHub workflow. Every commit, pull request, code review, and issue earns you XP. Complete daily quests, defeat weekly bosses, collect loot, craft items, and compete with your team on leaderboards.

## How It Works

```
Push code → GitHub webhook → Rook backend → XP + loot → CLI displays progress
```

1. **Connect** your GitHub repos with `rook connect`
2. **Code** as you normally would
3. **Check** your progress with `rook stats`, `rook dungeon`, or `rook leaderboard`
