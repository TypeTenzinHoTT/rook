# Turn Developer Momentum Into a Game Loop

Rook is an open-source CLI plus a self-hostable backend that watches GitHub activity and turns it into progression.
Commits, PRs, reviews, and issues become XP. Daily and weekly boards create goals. Loot, crafting, guilds, and prestige add longer-term loops. Socket.io pushes live updates back into the terminal while you work.

## How The Loop Works

```mermaid
flowchart LR
  D[Developer ships code] --> G[GitHub event]
  G --> W[/POST /api/webhooks/github]
  W --> B[Rook backend]
  B --> P[(PostgreSQL)]
  B --> S[Socket.io events]
  S --> C[Rook CLI]
  C --> D
```

1. You authenticate the CLI with a GitHub token and point it at a Rook backend.
2. `rook connect` creates GitHub webhooks for repos you manage.
3. GitHub events hit the backend and award XP plus quest progress.
4. XP bonuses from quest streaks, guild membership, and prestige are applied.
5. Loot, crafting, leaderboard, and battle events are emitted over Socket.io.
6. The CLI renders your current state through commands like `rook stats` and `rook leaderboard --watch`.

## Core Systems

<CardGroup cols={2}>
  <Card title="Progression" icon="arrow-up-right">
    Level = `floor(sqrt(totalXp / 1000)) + 1`. The curve gets steeper as total XP grows, which makes streak and multiplier systems matter.
  </Card>
  <Card title="Quests" icon="sword">
    Five daily quests and three weekly boss quests are materialized in PostgreSQL and updated from GitHub activity.
  </Card>
  <Card title="Loot & Crafting" icon="gem">
    Seeded loot items, a luck meter, and three base recipes create a light crafting economy on top of work activity.
  </Card>
  <Card title="Social" icon="users">
    Friends leaderboards, guild bonuses, guild quests, notifications, and share links turn solo usage into a team loop.
  </Card>
</CardGroup>
