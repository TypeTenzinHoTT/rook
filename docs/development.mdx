  **Prerequisites**: Node.js 18+, PostgreSQL 14+
</Info>

## Local Setup

<Steps>
<Step title="Clone the repository">

```bash
git clone https://github.com/TypeTenzinHoTT/rook.git
cd rook
```

</Step>

<Step title="Install CLI dependencies">

```bash
npm install
```

</Step>

<Step title="Build the CLI">

```bash
npm run build
```

</Step>

<Step title="Set up the backend">

```bash
cd backend
npm install
```

Create a `backend/.env` file:

```bash
DATABASE_URL=postgres://user:pass@localhost:5432/rook
PORT=4000
GITHUB_WEBHOOK_SECRET=your_secret_here
# Optional: OPENAI_API_KEY=sk-... (enables coach tips in rook stats)
```

</Step>

<Step title="Initialize the database">

Run the schema and migrations against your PostgreSQL instance:

```bash
psql $DATABASE_URL -f schema.sql
for f in migrations/*.sql; do psql $DATABASE_URL -f "$f"; done
```

</Step>

<Step title="Start the backend">

```bash
npm start
```

The server runs at `http://localhost:4000`.

</Step>

<Step title="Point the CLI at local backend">

```bash
rook login --api-url http://localhost:4000/api
```

</Step>
</Steps>

## CLI Commands Reference

| Command | Description |
|---------|-------------|
| `rook init` | Full onboarding: login + connect repos |
| `rook login` | Authenticate with GitHub PAT |
| `rook stats` | Level, XP bar, streak, achievements, coach tip |
| `rook dungeon` | Daily quests + weekly boss quests |
| `rook leaderboard [--friends] [--watch]` | Rankings with optional live refresh |
| `rook friends [list\|add\|remove]` | Manage your party |
| `rook inventory` | View loot drops |
| `rook craft` | Craft items from loot |
| `rook guild [create\|join\|invite\|leave\|stats]` | Guild management |
| `rook prestige` | Reset at level 20+ for permanent perks |
| `rook connect` | Set up GitHub webhooks |
| `rook history` | XP activity log |
| `rook share <target> [id]` | Share to Twitter/Discord/Slack |
| `rook notify` | Configure notification integrations |

## Project Structure

```
rook/
  src/             # TypeScript CLI source
    commands/      # Individual command handlers
    lib/           # API client, config, UI utilities
    types/         # TypeScript interfaces
  backend/
    server.js      # Express entry point
    routes/        # API route handlers
    lib/           # Business logic (XP, quests, loot, guilds)
    migrations/    # SQL migration files
    schema.sql     # Base database schema
  docs/            # Mintlify documentation
```

## Architecture

```
GitHub Webhooks → Backend (Express + PostgreSQL + Socket.io) → CLI (Commander.js)
```

- **Webhooks** capture push, PR, issue, and review events
- **Backend** calculates XP, manages quests, handles loot RNG, and broadcasts updates
- **CLI** displays progress with styled terminal output (chalk, boxen, ora)
- **Socket.io** powers real-time leaderboard updates when using `--watch`

## Documentation Development

To preview docs locally:

```bash
npm i -g mint
cd docs
mint dev
```

Preview at `http://localhost:3000`.
